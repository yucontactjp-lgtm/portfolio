# 生貝優志 ポートフォリオ

中小企業・個人事業主向けに、AIで業務を効率化するサービスを紹介するポートフォリオサイトです。

## サービス内容
- AIエージェント構築
- 業務自動化
- AIチャットボット構築
- HP制作

## 構成
| ファイル | 内容 |
|---|---|
| `index.html` | 1ページ縦長スクロール型のHTML |
| `styles.css` | スタイル（ダークネイビー×シアン配色） |
| `main.js` | フェードイン・カウントアップ・FAQ開閉・モバイルナビ |
| `preview-server.cjs` | ローカル確認用の簡易静的サーバー |

## ローカルでの確認
```bash
node preview-server.cjs
# http://localhost:4173 を開く
```

静的サイトのため、`index.html` をブラウザで直接開いても確認できます。

## デプロイ
Vercel など静的ホスティングにそのまま公開できます。
